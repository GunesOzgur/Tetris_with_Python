# Tetris_with_Python
Tetris game written via Python

This project consists of 6 modules and 1 source folder
These modules are:
  - TetrisMain
  - Board
  - GameOver
  - MainMenu
  - ScoreTable
  - Tetromino

TetrisMain is the main module.

"/src" is the source file. It contains images that are used in the game.
